Ohjelman käyynistäminen:

Ohjelman voi suorittaa Visual Studio Community 2019 versiolla 16.4.5
Ohjelman pääsee käsiksi avaamalla SaaTutka.sln:n Visual Studiolla ja suorittamalla sen.
SaaTutka.sln löytyy hakemiston juuresta.

API avain tarvitaan kontrollerissa. Tarkka polku on siis SaaTutka/Controllers/HomeController.cs
avainta kysytään rivillä 18.
